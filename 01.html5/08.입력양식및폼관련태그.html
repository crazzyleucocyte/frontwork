폼 관련 태그

form 태그내에 submit 버튼 글릭시 해당 form안에 작성된
사용자가 입력한 값들을 서버로 전송하면서 요청하는 역할 수행

form action="서버로 전송될 값" method="전송 방법"

*action 속성 : 해당 form에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로를 넣어줌 *method 속성 : 요청 전송 방식을 지정하는 속성(기본값은 get | post)
>get방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (url에 내가 검색한 검색어가 뜰 경우)
>post방식 : 요청시 사용자가 입력한 값들이 url에 노출이 되지 않는 방식(아이디나 비번같이 중요한 정보는 url에 노출되면 안된다.) form 내의 submit 버튼 클릭시 action에 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨(사용자가 입력한 값들은 key=value 세트로 전달)

search?id=kkk&pw=1234&age=24(id가 key, kkk가 value값이다) 웹에서 입력한 값들이 서버나 db로 넘어갈때 위의 양식으로 넘어간다.

첨부파일 :

type="hidden"

- 특정값을 화면에 노출시키지 않고 서버로 전송해야하는 정보가 있을때 사용 (사용자에게 보여주지 않아야 하는 정보)

type="submit | reset | button"

- submit : 서버로 form 안의 들어있는 모든 값들을 전송 - reset : form 안의 값들을 초기화 - button : 일반버튼(사용자 정의 버튼으로 자바스크립트에서 사용)

button[type="submit | reset | button"]
