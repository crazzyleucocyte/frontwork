이미지 관련 태그

* src속성

* alt 속성

-사진의 경로가 잘못되었거나 이미지를 제대로 표현하지 못할 경우 대체택스트의 용거 - 시각장애인들을 위한 스크린리더(화면낭독기)에서 이미지를 읽어주는 설명 문구


* width와 height의 속성

이미지의 가로, 세로 길이를 조걸하는 속성
고정길이(px), 가변길이(%)로 지정 => 기본값은 px단위

고정길이단위(px) == 기본값 : 화면 사이즈가 조정되어도 이미지의 크기는 변하지 않는다.


가변길이(%) : 화면사이즈 또는 부모요소사이즈에 따라 이미지의 크기도 같이 변함
