flex

기본 축을 기준으로 정렬

        *부모요소 : flex-container
         - display: flex
         - flex-direction: row(기본값), | column | row-reverse | column-reverse      
         - justify-content: 메인축 정렬(flex-direstion: row이면 row, column이면 column) 
         - align-items: 크로스축 정렬(flex-direstion: row이면 column을 어떻게 정렬 해줄것인지)
         - flex-wrap: 줄바꿈 여부
         - align-content: 여러줄인 경우의 크로스축 설정

        *자식요소(정렬의 주체) : fles-item
         - flex-grow : 빈 여백을 아이템들의 주어진 비율대로 늘어나면서 레이아웃 영역을 채움
         - flex-shrink : 넘치는 아이템 영역을 분배해서 레이아웃 영역 안에 아이템들이 배치 되도록
         - flex-basis : 아이템의 크기를 직접 설정
    
    flex : 파일을 웹에서 열어서 F12를 눌러서 스타일의 해당 flex코드를 선택 후 flex옆의 네모 모양 아이콘을 
    눌러서 원하는 모양이 됐을때 해당 코드를 복사해서 html에 붙여넣기 하면 된다.
