목록 관련 스타일(list~)

list-style-type : 블릿기호를 변경시켜줄 때 사용

list-style-image : 블릿기호로 이미지를 적용할 떄 사용

list-style-position : 블릿기호의 위치를 조정할 때 사용

        *순서가 없는 목록(ul)
        선택자 {
            list-style-type : desc(기본값, 까만 동그라미) | circle(빈 동그라미) | squar(네모) | none;

        }

        *순서가 있는 목록(ol)  잘 사용하지 않고 그냥 type="i" or type="a"이렇게 쓰는게 편하니깐 이렇게 쓰자
        선택자{
            list-style-type : decimal(기본값) | decimal-leading-zero;
            list-style-type : lower-alpha | upper-alpha
            list-style-type : lower-roman | upper-roman
        }
