font-size: 글꼴크기를 변경할떄 사용

        선택자{font-size : 크기(px|em|%)}
            - px : 픽셀단위로 고정크기
            - em : 부모요소 글꼴그기의 배수(1.5em)
            - rem : html 태그의 폰트 크기의 비율로 계산(1.5rem)  div의 폰트를 1.5 rem으로 했을떄 기본 div의 폰트 크기의 1.5배가 된다.
            - % : 상위요소 폰트크기의 비율로 계산(150%)

            -vw, vh : 현재 보이는 장의 크기에 따라 값이 변한다.(창의 width : 1000px -> 1vw : 10px)
        옵션들

        *고정값
        xx-small : 10px
        x-small : 10px
        small : 13px
        medium : 13px(기본값)
        larger : 18px
        x-larger : 24px
        xx-large : 32px

        *상대값
        larger : 부모의 글자크기의 120%
        smaller : 부모의 글자크기의 1/6
  • 고정크기 px 테스트
  • 가변크기 em크기 테스트
  • 가변크기 %테스트

font-weight : 글꼴의 두꼐 지정

    선택자{
        font-weight : normal | bold | bolder | lighter | 100 | 400 | 900;
    }
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 월래 굵기보다 더 가늘게

font-variant : 영문텍스트 문구를 작은 대문자로 변경할 떄 사용

    선택자 {
        font-variant : small-caps;
    }
