텍스트 관련 스타일(text~)

1. color : 텍스트의 색상 지정

        선택자 {
            color : 색상명 | 16진수 | rgb(x, x, x) | rgba(x,x,x,x)(투명도가 포함되어있음) | hsl(x,x,x) | hsla(x,x,x,x)
            *16진수 : #ff11 등등
            *rgb : rgb(144,255,0) 등등
            *rgba : (4번째 자리 값 투명도 0~1)
        }

2. text-decoration : 텍스트에 줄을 긋거나 줄을 없ㅇ앨때 사용

        선택자{
            text-decoration : non | underline | overline | line-through;
        }

  • 텍스트 영역 위에 줄 긋기

3. text-transform : 영문 텍스트의 대소문자 변환

        선택자 {
            text-transform : uppercase | lowercase | capitalize;
        }
  1. uppercase : 모든 영문자를 대문자로
  2. LOWERCASE : 모든 영문자를 소문자로
  3. capitalize : 영문자의 first text 대문자로

4. text-shadow : 텍스트에 그림자 효과를 줄때 사용

        선택자 {
            text-shadow : 가로거리(x) 세로거리(y) [번짐정도] [색상];
        }

        선택자 {
            text-align : left(기본값) | right | justify | center;
        }
